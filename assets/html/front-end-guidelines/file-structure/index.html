File Structure

sass/style.scss @imports the individual .scss files and compiles them (via the gulpfile.js) into assets/css/style.css.

The Ushahidi Platform includes 'right to left' support. The RTL styles are located assets/css/rtl-style.css.

Influenced by Brad Frost's Atomic Design, we created our own version. Read below for more detail.

The Ushahidi Platform uses a Sass mixin library called Bourbon. This library provides fast and easy solutions to writing complex css. Visit the Bourbon docs for examples.

Bourbon is installed via Bower and is imported via sass/style.scss.
