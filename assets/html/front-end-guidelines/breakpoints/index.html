Breakpoints & Media Queries

The Ushahidi Platform was designed using a mobile-first approach, meaning we approached the design with smaller devices in mind, then added components and adjusted layouts as the screen size increased. This approach allows the app to function and convey the appropriate information regardless of the user's device. It works on mobile phones and desktop computers alike.
