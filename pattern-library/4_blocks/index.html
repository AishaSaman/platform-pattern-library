Mainsheet

A mainsheet sits on top of the existing layout so that a user can access content without navigating away from the page they are on. This pattern is typically used to present users with a group of choices, like which survey they'd like to respond to, which field type they'd like to add, etc.

It leverages _toggle.js for the toggling of its display within layouts.

Modal

A modal sits on top of the existing layout so that a user can access content without navigating away from the page they are on. This pattern is typically used to assist users in managing more detailed aspects of the content they're viewing on the current page. For example, configuring a survey field or adding a response to a collection.
